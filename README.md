# Lectorem

## Welcome to **Lectorem**, RSS Feed Reader Apps !

## Features

*   Filtering RSS Feed By Priority
*   Adding New RSS Feed
*   Replacing or Deleting RSS Feed
*   Show Complete Article

## Platforms

Lectorem has been used on a unique plateform :

*   Android OS
